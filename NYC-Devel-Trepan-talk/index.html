Features

  • syntax highlighting of Perl code
  • context-sensitive command completion
  • extensive online-help
  • gdb syntax
  • out-of-process and remote debugging
  • interactive shell support
  • code disassembly
  • extensibile
  • comes with extensive tests
  • is not as ugly as perl5db

Opinion

Most common criticism I get of Perl is its one-offness:

  • %, @ sigils
  • vars like $^X, @_, or $<

A debugger for Perl doesn't have to be so one-off.
